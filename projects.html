Implementing classical search algorithms

Mazes

In order to practice for my artificial intelligence class, I implemented an automated maze generation + resolution using classic search algorithms on Python. Graphical display was implemented using the Pygame library.

During this project, I also got to discover the amazing world of maze generation algorithms

Implemented search algorithm : Depth-First Search

Implemented maze generation algorithms : Prim's algorithm & Recursive Backtrack algorithm

You can find this project here

Grid size : 70*70
Maze generation algorithm : Prim's
Search algorithm : Depth-First Search
The gray areas are the ones visited by the Search Algorithm while it tries to find the exit
The blue line is the way found by the search algorithm
